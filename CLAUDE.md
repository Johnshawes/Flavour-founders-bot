# Flavour Founders Instagram Bot

## Trigger Keywords

These keywords trigger an automatic DM when someone comments them on a post.
Edit this list to add or remove trigger words — the bot reads them on startup.
Each keyword is followed by its funnel type: `application` or `lead_magnet`.

### Application Funnel (high intent — qualification flow → discovery call)
- info | application
- interested | application
- how much | application
- tell me more | application
- sign me up | application

### Lead Magnet Funnel (curious/early stage — deliver value → soft sell)
- system | lead_magnet
- freedom | lead_magnet
- margins | lead_magnet
- calculator | lead_magnet
- free | lead_magnet
- guide | lead_magnet
- help | lead_magnet
